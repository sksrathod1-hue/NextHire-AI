# NextHire-AI# 🚀 NextHire AI

**NextHire AI** is an AI-powered interview preparation platform designed to help job seekers prepare smarter and get hired faster.

It provides intelligent resume analysis, ATS score evaluation, AI-generated mock interviews, personalized interview feedback, and job application tracking—all in one modern platform.

---

## ✨ Features

* 📄 AI Resume Analyzer
* 🎯 ATS Score Checker
* 🤖 AI Mock Interview
* 💬 AI Interview Feedback
* 📊 Career Dashboard
* 📈 Progress Analytics
* 💼 Job Application Tracker
* 📅 Interview Scheduler
* 👤 User Authentication
* ⚙️ Admin Dashboard
